# ПРОМТ ДЛЯ HEYGEN

Создай обучающее видео по теме «[ТЕМА]» на русском языке.

**Источники:**
- Используй прикреплённую презентацию ([N] слайдов) — показывай слайды по порядку
- Используй прикреплённый сценарий — в нём текст диктора для каждого слайда

**Аватар:**
- Аватар присутствует во всех сценах
- На слайдах — аватар в углу (picture-in-picture), не перекрывает контент
- Первая и последняя сцена — аватар крупным планом на нейтральном фоне

**Озвучка:**
- Голос: русский, профессиональный тон лектора
- Произношение аббревиатур строго по словарю (прикреплён)

**Запрещено:**
- Фоновая музыка
- Добавление информации, которой нет в сценарии
- Пропуск слайдов или изменение порядка

**Субтитры:** включить автоматические на русском языке.
**Качество:** 1080p.
**Переходы между сценами:** плавные (dissolve).
